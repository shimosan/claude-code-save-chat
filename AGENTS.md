# Codex Repository Instructions

The canonical repository maintenance rules are in `.claude/CLAUDE.md`.

Before normal development or maintenance work in this repository, read
`.claude/CLAUDE.md` and follow it as repository instructions. Interpret
references to "Claude Code" as describing the distribution target or historical
naming, not as a requirement to use Claude-specific tooling.

Do not treat `dotclaude/CLAUDE.md` or `dotcodex/AGENTS.md` as project
instructions by default; they are distribution masters for end-user Claude Code
and Codex configuration.
